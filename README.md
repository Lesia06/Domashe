##Сформулювати своє розуміння, що таке тип даних.
*Тип даних - це спеціальна характеристика об'єкта програми, яка визначає його структуру та формат зберігання даних. Тип даних визначає, які значення можуть бути присвоєні об'єкту і яким чином вони будуть оброблятися та використовуватися в програмі. Наприклад, цілочисельний тип даних дозволяє присвоювати лише цілі числа і проводити з ними операції додавання, віднімання, множення тощо. Рядковий тип даних використовується для збереження символьних рядків та дозволяє проводити операції конкатенації та порівняння символів.*

##Сформулювати своє розуміння типізації - динамічної, статичної, сильної та слабкої.
*Статична типізація передбачає, що тип змінної фіксується на етапі компіляції програми і не може бути змінений під час виконання програми. Такі мови програмування, як Java та C++, використовують статичну типізацію. Якщо в програмі використовується змінна певного типу, то вона не може бути використана як змінна іншого типу.

Динамічна типізація передбачає, що тип змінної може змінюватись під час виконання програми. У мовах програмування з динамічною типізацією, таких як Python та Ruby, змінні можуть змінювати тип під час виконання, що надає розробникам більшу гнучкість. Однак, така гнучкість може призвести до помилок під час виконання програми.
Сильна типізація передбачає, що мова програмування не дозволяє автоматичного перетворення типів даних без явного вказівника.
Слабка типізація передбачає, що мова програмування автоматично перетворює типи даних виразів, які взаємодіють між собою, якщо це необхідно.*
